整套「文章开发.EXE」分为 4 大模块任务，搭配标准 25min 专注 + 5min 休息，4 轮后长休 20min，精准匹配增量式、模块化开发节奏


一、写文章（论文核心，耗时最长）
单番茄：25min 写稿 / 查文献｜5min 休息
任务拆分番茄轮次：
番茄 1：确定选题 idea，梳理研究方向框架
番茄 2–4：搜集参考文献，筛选、整理文献笔记（要求≥40 篇，3 轮集中检索）
番茄 5–8：结合 idea + 文献撰写初稿，分段完成 5000 字主体内容
番茄 9–10：完善初稿，补全逻辑、案例、论证
番茄 11：通读全文优化语句、修正逻辑
番茄 12：终审核对格式、参考文献、查重调整


二、画图（配图制作）
单番茄：25min 绘图 / 写提示词｜5min 休息
番茄 1：搜集同类优质配图，总结风格、构图关键词
番茄 2：根据论文全文，批量生成配图提示词清单
番茄 3–5：输入提示词 AI 生成图片，多轮对话迭代优化配图
番茄 6：图文匹配，调整图片尺寸、标注，嵌入文稿对应位置


三、作者信息 & 投稿包整理
单番茄：25min 整理信息 / 打包文件｜5min 休息
番茄 1：完善信息表（姓名、单位、邮箱、联系方式等）
番茄 2：把作者信息插入论文对应位置，统一格式
番茄 3：按期刊 / 公众号要求调整版式，打包生成投稿 zip 文件


四、投自媒体（小红书 / 公众号 / 知乎）
全程 AI 驱动自动化操作，25min 一轮：
番茄 1：选定发布平台，梳理发布规范
番茄 2：AI 驱动批量注册登录，自动收验证码、人机验证、页面操作
番茄 3：AI 自动巡检发布页面，识别报错、处理问答 / 审核问题


五、整体每日高效循环模板
上午 4 轮番茄：主攻【写文章】核心内容，结束长休 20 分钟
下午前 2 轮：【画图】配图制作；后 2 轮：【作者信息 + 投稿包】整理
晚间 1–2 轮：【自媒体投稿】操作、查漏补缺
收尾检查 1 个番茄：核对三项硬性要求：5000 字以上、图文搭配、参考文献齐全


适配长任务的番茄小技巧
写文献、写长初稿容易分心，中途中断直接作废当前番茄，重新计时
画图、AI 生成图片属于创意工作，休息时远眺放松，不要刷短视频
模块化拆分，严格遵循增量式开发，一个番茄只完成一小块细分任务，不跨模块混做
每完成 4 个完整番茄，必须长休息 15–20 分钟，避免长时间写论文视觉、脑力透支!
